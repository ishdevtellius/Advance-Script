# Advance-Script

Automation scripts for Kaiya (Tellius) Text2SQL evaluation and related workflows.

## Source Truth

The **Source Truth** folder contains a CLI to populate and evaluate natural-language queries against the Kaiya API using a Google Sheet as the source of truth.

- **Setup & usage:** see [SOURCE_TRUTH.md](SOURCE_TRUTH.md).
- **Contents:** `Source Truth/` — config, Kaiya client, evaluator, sheet utils, data-entry and query-evaluation flows.
